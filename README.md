# 校园生活服务平台前台项目
# 项目依赖包版本信息如下：

|    依赖    |  版本  |
| :--------: | :----: |
|    vue     | 3.2.45 |
| uview-plus | 3.3.5  |
|   pinia    | 2.0.36 |
|    sass    | 1.56.1 |

# 使用 yarn 安装依赖

```
yarn
```

一定要使用 yarm，否则 yarn.lock 就没用了，那你的启动依然会报错。  
然后执行相应平台的启动命令即可。
我的开发 node 版本是：16.20.2，如果环境异常使用 nvm 切换到这个版本尝试一下。
